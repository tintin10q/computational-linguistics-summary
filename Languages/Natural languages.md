# Natural languages 

Natural languages are languages that are:
- Conventional 
- A set of related systems
- There is redundancy 
- Subjected to changed a lot
- Context dependent. 
- Ambigious

It in't 
- Formal logic
- A Programming language
- A Machine language 

## Examples 

Examples of natural langauges are:
- English 
- Dutch
- Spanish 
- Japaneese
- Chineese
- German 


Natural languages often follow a [[Zipfian Distribution]].
